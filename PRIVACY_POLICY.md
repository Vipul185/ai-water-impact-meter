# AI Impact Meter Privacy Policy

Last updated: April 1, 2026

AI Impact Meter estimates electricity and cooling-equivalent water usage for prompts typed into supported AI websites.

## What the extension reads

The extension reads text that the user types into supported AI chat composers so it can estimate prompt impact before or when a message is sent.

Supported sites currently include:

- ChatGPT
- Claude
- Gemini
- Perplexity
- DeepSeek
- Grok
- Microsoft Copilot

## What data is stored

AI Impact Meter stores only aggregate usage counters in the browser's local extension storage, including:

- per-chat estimated electricity and water totals
- per-site estimated electricity and water totals
- combined estimated totals across supported sites
- request counts associated with those totals

AI Impact Meter does not intentionally store full prompt text in persistent storage.

## What data is sent to the developer

AI Impact Meter does not send prompt text, page content, browsing history, or usage counters to the developer's servers.

AI Impact Meter does not use analytics, advertising trackers, or remote logging.

## How data is used

The data read from supported pages is used only to:

- estimate the current prompt's electricity and water impact
- update local aggregate counters
- display those estimates to the user inside the extension popup and on supported pages

## Data sharing

AI Impact Meter does not sell, share, or transfer user data to third parties.

## Data retention and deletion

Aggregate counters remain in the browser's local extension storage until the user removes the extension or the extension provides a reset feature.

## Security

AI Impact Meter runs locally in the browser and bundles its code inside the extension package. It does not load remote executable code.
